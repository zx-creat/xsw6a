Client：一些docker的命令。
Docker daemon：负责docker执行命令。

简单使用

docker run ubuntu echo xsw6a:使用docker在ubuntu这台机器上(docker开启的Ubuntu服务)执行命令 echo xsw6a。

docker images：查看当前存在的镜像。

docker ps：查看正在运行的镜像。
docker stop：停止正在运行的镜像。
docker rmi：删除镜像。
